Project Summary

The said project was requested by the client behalf of Sri Sathya Sai District Police, AP.

The project has two parts:

  1. ssspolice website
  2. sssd-portal webapp

ssspolice website

The said website is built using HTML, CSS, and Javascript and the same has been adapated to WordPress. This has one landing page and everal content pages.

sssd-portal

The sssd-portal is a webapp which will be hosted in the subdomain of the above website. The primary purpose of this webapp is to record the day-to-day assist the police personnel during Election 2024.

Additionally, there are other significant features of the webapp:

  1. Admin Dashboard: which presents the day-to-day updates of various enforcements like cash seizure, liquor seizure, NDPL-DPL, NBW, arms recovery, drugs recovery, freebies recovery. Also an embedded map of the district.
  2. Village information page: This helps the user to find a particular village in the entire district using the list-navigation mehod. The resultant page displays the information about the village, PIPs and Poltical Leaders of the village and assigned polling booths. Further, this redirects to dedicated PIP, Political Leaders and Poll Booth information page where user gets the detailed report of respective entities.
  3. Polling Booth: This page designed to find a particular polling booth using dependent dropdown menu
  4. Enforcement: This page is dedicated to see the day-to-day enforcement activities as well as the cumulative of respective activities. Data for these entities will be frnished by the enforcement department using forms, which is discussed in detail in the forms section.
  5. Force Deployment: This page represents force deployment for the year 2019 and 2024. The 2019 mostly be a PDF while 2024 will be the data the enforcement department will furnish couple of days before elections.
  6. Training: Training Details
  7. Route Maps: This page will contain various routemaps required for the department
  8. Checkposts: This page divided into inter-state and intra-state listing various checkposts in the district with its embedded google maps.
  9. Forms: This section contains several forms to help the user fill the data to the database. The entered data will be logically rendered in the portal where and when required.
    • Annexure - III: The data entered in this form will reflect in the statistic panels in admin home page
    • Booth: The data enetered in this form appears in Poll Booth information both in Village information page as well as Poll Booh dedicated page
    • Deployment: The data enetered in this form will appear in Poll Booth Details page
    • Leaders: The data enetered in this form will appear in Village Information Page
    • LOR: The data entered in this form will reflect in the statistic panels in admin home page
    • PIP: The data enetered in this form will appear in Village Information Page
    • Village: The data enetered in this form will appear in Village Information Page
